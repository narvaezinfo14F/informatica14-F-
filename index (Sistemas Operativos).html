Sistemas operativos

¿Qué es un sistema operativo?

Un sistema operativo (SO) es un software que actúa como intermediario entre el hardware de una computadora y el usuario. Su principal función es gestionar los recursos del hardware y permitir que otras aplicaciones se ejecuten. Sin un sistema operativo, los dispositivos como computadoras, teléfonos inteligentes o tabletas no podrían funcionar de manera eficiente.

Algunos de los objetivos de los Sistemas Operativos son los siguientes:

• Acercar el sistema computacional al usuario facilitando su uso.

• Usar el hardware de una manera eficiente

• La mejor forma de entender un sistema operativo, es entender como se construye

• Ayuda a comprender en tener una visión histórica de su desarrollo

Algunos tipos de Sistemas Operativos son:

• Mono usuario y multiusuario

• Monotarea y multitarea

• Multiprocesamirnto o distribuidos

• Procesamiento por lotes o de tiempo compartido

• De red y cliente

El hadware se refiere a todos los componentes físicos de un sistema informático. Esto incluye elementos como:

• Teclado

• Mouse

• Escáner

• Micrófono

• Monitor

• Impresora

• Proyector

• Memoria USB

• Módem

• Ruter

• Disco duro

El software es un conjunto de programas, datos y procedimientos que permiten realizar tareas en un sistema informático. Este es intangible y es el que da instrucciones al hardware para que funcione correctamente.

Algunos softwares son:

• Microsoft office

• Google docs

• Zoom

• macOS

• Libre Office

• Adobe Photoshop

• Adobe ilustrador

• Visual studio

• Windows

• Gestores de bases de datos

• Teams

Fnciones de sistemas operativos

• Administrar la memoria: El sistema operativo gestiona la memoria RAM y asigna los recursos necesarios a las aplicaciones que se ejecutan

• Ejecutar problemas: Permite que los usuarios ejecuten los programas que necesiten

• Gestionar el hadware: Lo gestiona incluyendo la tarjeta de red, lo que permite a la red a internet

• Interacción usuario: Proporciona interfaces gráficas de usuario (GUI) o de línea de comandos para que los usuarios interactúen con la computadora.

• Administrar archivos: Garantiza la seguridad, administra las conexiones de red, direccional las entradas y salidas de datos; lo hace a través de periféricos de entrada o salida.
